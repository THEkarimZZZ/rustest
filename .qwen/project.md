# Project: TestFlow (Educational Platform)

## Обзор
Платформа для создания тестов и управления учебными классами. Основной фокус на строгом академическом дизайне, мобильной доступности для учеников и глубокой аналитике для преподавателей.

## Дизайн-система (Strict & Modern)
- **Стиль**: Минималистичный "Enterprise" дизайн. Использование `shadcn/ui`.
- **Темы**: Светлая и Темная (с поддержкой системных настроек).
- **UX**: Адаптивность 100%. Прохождение теста оптимизировано под управление одной рукой (мобильные устройства).
- **Анимации**: Микро-взаимодействия (hover, плавные переходы между вопросами), не отвлекающие от сути.

## Юридическая и техническая безопасность
- **Compliance (РФ)**: Соответствие ФЗ-152. Обязательное согласие на обработку ПД и принятие Пользовательского соглашения (User Agreement) с активными ссылками при регистрации.
- **Анти-фрод**: 
    - Ограничение на создание нескольких аккаунтов (Rate Limiting, проверка по email).
    - Базовая защита от списывания (детектирование потери фокуса окна браузера).

## Ключевой функционал
- **Конструктор**: Разные типы вопросов, блоки с текстом/подсказками.
- **Градация оценок**: Автоматический перевод % баллов в оценки (2, 3, 4, 5) по настраиваемой шкале учителя.
- **Аналитика**: Графики качества знаний, среднего балла и экспорт результатов в Excel.
