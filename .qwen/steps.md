# Implementation Roadmap

## Шаг 1: Юридический фундамент и Auth
- [ ] Инициализация проекта и подключение Supabase.
- [ ] Создание UI регистрации: поля ФИО, Роль, Email.
- [ ] **Legal Block**: Добавление чекбокса с ссылками на `/privacy-policy` и `/terms`. Блокировка кнопки "Регистрация" до активации чекбокса.
- [ ] Настройка записи `legal_accepted_at` в профиль пользователя.

## Шаг 2: Кабинет Учителя и Классы
- [ ] Создание интерфейса "Мои классы".
- [ ] Логика генерации `invite_code` и ссылка-инвайт для учеников.
- [ ] Функционал управления учениками: переименование, удаление из класса.
- [ ] Базовая страница статистики класса (средний балл).

## Шаг 3: Конструктор тестов и Настройка оценок
- [ ] Интерфейс добавления вопросов разного типа.
- [ ] **Модуль оценивания**: Слайдер или инпуты для настройки порогов (например, "Оценка 5 от 90%").
- [ ] Реализация блоков "Подсказка" и "Текст" (без баллов).
- [ ] Предпросмотр теста (Mobile View).

## Шаг 4: Прохождение теста (Student Flow)
- [ ] Dashboard ученика со списком доступных тестов.
- [ ] Интерфейс тестирования: строгий таймер, индикатор прогресса.
- [ ] Экран завершения: автоматический расчет % и показ оценки (на основе шкалы учителя).

## Шаг 5: Аналитика и Экспорт
- [ ] Создание таблицы результатов для учителя.
- [ ] Интеграция графиков (распределение оценок в классе).
- [ ] Кнопка "Экспорт в Excel" (формирование файла с колонками: ФИО, Тест, Балл, %, Оценка).
- [ ] Финальная полировка темной темы и безопасности (RLS политики).
